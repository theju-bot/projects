Q&A JavaScript

Here are some of the questions

What is JavaScript?

JavaScript is a programming language used to make web pages interactive.

What is the difference between var, let, and const?

var is function-scoped, let and const are block-scoped; const can't be reassigned.

What is DOM in JavaScript?

The DOM (Document Object Model) is a structured representation of the HTML that JavaScript can manipulate.

What is an event listener?

An event listener waits for user actions (like clicks or keypresses) and runs code in response.

What is the difference between == and ===?

== checks value only, === checks value and type.
